üåê Web Browser (Chrome/Firefox/Edge)
Built-in developer tools for analyzing websites and investigating phishing

üìå Investigating Phishing Websites

Step 1: Inspect URL Without Clicking

Hover over suspicious links to preview destination URL in bottom-left corner.

Example: Email shows "www.sbi.co.in" but hovering reveals "sbi-secure-login.tk"
Step 2: Check SSL Certificate

Click padlock icon in address bar to view certificate details.

Example: Legitimate SBI has certificate issued to "State Bank of India". Fake shows "Domain Privacy Service"
Warning: HTTPS doesn't guarantee legitimacy - fraudsters can get SSL certificates
Step 3: Analyze Domain Name

Look for typosquatting, extra words, unusual TLDs (.tk, .ml, .ga)

Examples: Legitimate: amazon.com | Fraudulent: amazom.com, amazon-india.tk
Step 4: Use Developer Tools (F12)

Right-click ‚Üí Inspect or press F12

  • Network tab: See all server requests and data transmission
  • Console tab: Check for JavaScript errors or suspicious scripts
  • Sources tab: View loaded scripts and identify malicious code
Step 5: Check Website Reputation
Recommended Tools:
  • VirusTotal (virustotal.com) - Scan URL with 70+ security engines
  • URLVoid (urlvoid.com) - Check domain reputation
  • Whois (whois.com) - Domain registration info
  • Google Safe Browsing - Check if site is flagged

üìå Verifying Social Media & Profile Images

Google Reverse Image Search

Right-click image ‚Üí "Search image with Google" OR visit images.google.com ‚Üí camera icon

Example: Profile claims "Dr. Rahul Sharma from Delhi" but image belongs to US model
TinEye Alternative

Visit tineye.com and upload image - shows oldest version with timestamps

Check Profile History
  • Facebook: About ‚Üí Contact Info ‚Üí Join date
  • LinkedIn: Scroll to "Member since" date
Red Flag: Account under 90 days old, minimal posts, few connections

üìå Analyzing Email Headers to Trace Origin

Step 1: Install & Setup

Download from thunderbird.net ‚Üí Add your email account

Step 2: View Email Headers

Open email ‚Üí More (three dots) ‚Üí "View Source" or Ctrl+U

Step 3: Analyze Critical Headers
Key Fields:
  • From: Display email (easily spoofed)
  • Return-Path: Actual reply address (harder to fake)
  • Received: Complete email path (read bottom to top)
  • SPF/DKIM/DMARC: Authentication results
  • X-Originating-IP: Sender's IP address
Example: From shows "noreply@sbi.co.in" but Return-Path is "scammer123@gmail.com"
Step 4: Trace IP Addresses

Extract IPs from Received headers ‚Üí Use whatismyipaddress.com/ip-lookup

Example: Email claims "HDFC Mumbai" but IP traces to Nigeria
Step 5: Verify Authentication
Interpretation:
  • ‚úì SPF Pass + DKIM Pass + DMARC Pass = Legitimate
  • ‚úó Any "fail" = High probability of spoofing
  • ? Missing authentication = Suspicious

üìå Safely Analyzing Email Attachments

Never Open Suspicious Attachments
Warning: Can install ransomware, keyloggers, malware
Scan with VirusTotal

Save attachment ‚Üí Upload to virustotal.com for 70+ antivirus scans

Check File Extension
Dangerous: .exe, .scr, .bat, .js, .vbs, .zip (password-protected), invoice.pdf.exe
üñºÔ∏è ExifTool - Metadata Analyzer
Extract metadata from images, videos, and documents

üìå Extracting Image Metadata

Installation
Windows: Download from exiftool.org ‚Üí Extract to folder
Mac: brew install exiftool
Linux: sudo apt-get install libimage-exiftool-perl
Basic Extraction
exiftool image.jpg

Shows: Camera model, GPS coordinates, date/time, software used, author

Verify Image Authenticity
exiftool -a -G1 suspicious_image.jpg
Example: Scammer claims "photo taken today" but EXIF shows "Create Date: 2019:05:15, Software: Photoshop"
Extract GPS Location
exiftool -gps:all image.jpg
Example: Romance scammer claims "London UK" but GPS shows Lagos, Nigeria (6.5244, 3.3792)
Batch Process Multiple Images
exiftool -r -csv -n folder_name > report.csv

Analyze all images in folder, export as CSV

Remove Metadata (Privacy)
exiftool -all= image.jpg

Strip all metadata before sharing (creates backup automatically)

üìå Analyzing Document Metadata

Extract PDF Metadata
exiftool document.pdf
Shows: Author, Creator software, Producer, Create Date, Modify Date
Detect Forged Documents
Example: Fake bank statement claims "HDFC Bank" but Author: "Raj Kumar", Creator: "Microsoft Word" (banks use specialized software)
üéß Audacity - Audio Analysis Tool
Analyze voice recordings, detect manipulation

üìå Analyzing Vishing Call Recordings

Installation

Download from audacityteam.org (free, open-source)

Legal Notice: Recording calls may be illegal in some jurisdictions. In India, one-party consent is generally acceptable.
Import Recording

File ‚Üí Open ‚Üí Select recording (MP3, WAV, M4A, etc.)

Enhance Audio Quality
Noise Reduction:
1. Select portion with only noise
2. Effect ‚Üí Noise Reduction ‚Üí Get Noise Profile
3. Select all (Ctrl+A)
4. Effect ‚Üí Noise Reduction ‚Üí Apply
Amplify Weak Audio:
Select all ‚Üí Effect ‚Üí Amplify ‚Üí OK
Normalize Volume:
Select all ‚Üí Effect ‚Üí Normalize ‚Üí Set to -1.0 dB
Detect Voice Manipulation
Spectrogram Analysis:
Track dropdown ‚Üí Spectrogram ‚Üí Look for unnatural frequency patterns
Voice changers show distinctive artifacts, horizontal lines, robotic patterns
Extract Key Segments

Select important part ‚Üí File ‚Üí Export ‚Üí Export Selected Audio

Add labels: Edit ‚Üí Labels ‚Üí Add Label at Selection

Analyze Background Sounds
Clues: Call center noise, traffic (urban), animals (rural), accent patterns

üìå Detecting Audio Tampering

Visual Waveform Inspection

Look for: Abrupt amplitude changes, unnatural silence gaps, discontinuities, click/pop sounds

Spectral Analysis

Track dropdown ‚Üí Spectrogram ‚Üí Look for vertical lines (cuts/splices), frequency pattern changes

üìå Creating Transcriptions for Evidence

Slow Down Playback

Effect ‚Üí Change Tempo (without pitch change) ‚Üí Reduce 10-30%

Add Timestamps

At key moments ‚Üí Edit ‚Üí Labels ‚Üí Add Label ‚Üí Type description

Export labels: File ‚Üí Export ‚Üí Export Labels

Export Enhanced Version

File ‚Üí Export ‚Üí Export as WAV (highest quality)

Important: Always keep original unedited recording. Enhanced version is supplementary only.
